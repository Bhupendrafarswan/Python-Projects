#  Time Series & Category Charts

##  Project Overview
This project analyzes sales data using:
- Line Chart (Time Series)
- Monthly Aggregation
- Bar Chart (Category Comparison)
- Pie Chart (Share Distribution)

##  Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

##  Output
Charts generated:
- line_chart.png
- monthly_chart.png
- bar_chart.png
- pie_chart.png

```bash
python project1.py
