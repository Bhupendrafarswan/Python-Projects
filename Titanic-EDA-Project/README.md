# Titanic EDA Project 

##  Objective
Perform Exploratory Data Analysis on Titanic dataset to find survival patterns.

##  Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

##  Analysis Done
- Data cleaning
- Missing value handling
- Survival analysis (Gender, Class, Age)
- Data visualization

## Key Insights
- Females survived more than males
- Higher class passengers had better survival chances
- Age had some impact on survival
